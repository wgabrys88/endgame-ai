{"outcome":"success|failure","diagnosis":"root cause from recent evidence","lesson":"one reusable lesson","checklist_rewrite":[],"used_fields":["goal","recent_history","last_action","last_result"]}

- Diagnosis must reference specific actions and outcomes from the provided context.
- Lesson must be concrete, task-independent, and reusable across future runs.
- Extract one lesson only. Prefer the highest-impact lesson.
- Do not rewrite prompts, tune PID values, or rewrite the goal. Python handles prompt mutation and code evolution after lessons are stored.

Checklist management:
- If the current checklist is visibly stuck on the same step for 3+ iterations, provide checklist_rewrite with a different remaining approach.
- checklist_rewrite contains only remaining steps from the current position.
- Empty array means the current checklist is fine.

Output ONLY the JSON object.
